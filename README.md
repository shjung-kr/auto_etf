# 🚀 ETF 자동 거래 프로그램

ETF 수익률을 자동으로 계산하고 기술적 지표를 분석하여 매도/매수 신호를 제공하는 Python 기반 자동 거래 프로그램입니다.

## 📋 주요 기능

### 1. **ETF 데이터 수집 및 분석**
   - Yahoo Finance API를 통한 실시간 ETF 가격 데이터 수집
   - 3개월, 1개월, 1주, 1일 등 다양한 기간의 수익률 계산

### 2. **기술적 지표 분석**
   - **이동평균선 (Moving Average)**
     - 단기 MA (20일) vs 장기 MA (60일)
     - 상향/하향 추세 판단
   
   - **RSI (Relative Strength Index)**
     - 70 이상: 과매수 (매도 신호)
     - 30 이하: 과매도 (매수 신호)
   
   - **MACD (Moving Average Convergence Divergence)**
     - MACD 선 & 신호선 교차로 추세 변화 감지

### 3. **자동 거래 신호 생성**
   - 복수 지표 기반 신호 생성 (BUY, SELL, HOLD)
   - 신호 신뢰도 계산 (0~100%)
   - 목표 수익률 달성 시 자동 매도
   - 손절 기준 초과 시 자동 청산

### 4. **자동 알림 기능**
   - 콘솔 알림
   - 이메일 알림 (Gmail 지원)
   - Discord 웹훅 알림

### 5. **포트폴리오 관리**
   - 매입/매도 이력 기록
   - 포지션별 수익률 추적
   - 포트폴리오 총 수익률 계산
   - 승률 및 성과 분석

## 🛠️ 설치 및 설정

### 1. Python 설치
Python 3.8 이상 필요

### 2. 의존 라이브러리 설치
```bash
pip install -r requirements.txt
```

### 3. 환경 변수 설정 (선택사항)

`.env` 파일을 생성하여 다음을 설정할 수 있습니다:

```
# Gmail 알림 설정
EMAIL_ADDRESS=your_email@gmail.com
EMAIL_PASSWORD=your_app_password

# Discord 웹훅
DISCORD_WEBHOOK=https://discord.com/api/webhooks/YOUR_WEBHOOK_ID
```

**Gmail 앱 비밀번호 생성 방법:**
1. Google 계정 보안 설정 (https://myaccount.google.com/security)
2. 2단계 인증 설정 후
3. "앱 비밀번호" 섹션에서 생성

**Discord 웹훅 생성 방법:**
1. Discord 서버의 채널 설정
2. "연동" → "웹훅" → "새 웹훅"

## 📈 주요 지표 설명

### 이동평균선 (Moving Average)
- **단기 이동평균 > 장기 이동평균**: 상향 추세 (매수 신호)
- **단기 이동평균 < 장기 이동평균**: 하향 추세 (매도 신호)

### RSI (상대강도지수)
- **70 이상**: 과매수 상태 (매도 고려)
- **30 이하**: 과매도 상태 (매수 고려)
- **30-70**: 중립 상태

### MACD (이동평균수렴확산지수)
- **MACD > 신호선**: 상향 신호 (매수)
- **MACD < 신호선**: 하향 신호 (매도)

## ⚠️ 주의사항

1. **백테스팅 필수**
   - 실제 거래 전에 코드 수정 후 충분한 테스트 필요

2. **위험 관리**
   - 목표 수익률과 손절 기준을 신중하게 설정
   - 최소 보유 기간을 고려하여 과도한 거래 방지

3. **API 제한**
   - Yahoo Finance는 무료이지만 요청 제한이 있을 수 있음
   - 과도한 요청 시 차단될 수 있으니 조절 필요

4. **시장 조건**
   - 기술적 지표는 과거 데이터 기반으로 100% 정확하지 않음
   - 시장 상황에 따라 신호가 실패할 수 있음

5. **자동 거래**
   - 현재 코드는 시뮬레이션용 (실제 주문 미포함)
   - 실제 거래 연동 시 증권사 API 연결 필수
