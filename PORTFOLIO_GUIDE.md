# 📊 자산 관리 가이드

자신의 포트폴리오에 자산을 추가하고 관리하는 방법을 설명합니다.

## 🚀 빠른 시작

### 방법 1: CLI 도구 사용 (권장) ⭐
가장 쉬운 방법입니다. 다음 명령을 실행하세요:

```bash
python manage_portfolio.py
```

메뉴에서:
1. **ETF 자산 추가** - 새로운 ETF 자산을 추가
2. **보유 자산 조회** - 현재 보유 자산 확인
3. **자산 수정** - 기존 자산 수정
4. **자산 제거** - 자산 삭제
5. **💰 현금 관리** - 현금 추가, 수정, 확인 (NEW!)
6. **포트폴리오 전체 보기** - 전체 포트폴리오 보기
7. **종료** - 프로그램 종료

### 💰 현금 관리 기능

포트폴리오에 현금을 함께 관리할 수 있습니다!

**메뉴에서 "5. 💰 현금 관리" 선택:**
- **현금 추가**: 초기 자금, 월급, 배당금 등을 추가
- **현금 확인**: 현재 보유 현금 확인
- **현금 수정**: 기존 현금액 수정
- **현금 제거**: 현금 기록 삭제

예시:
```
1. 초기 자금 $50,000 추가
2. 배당금 $500 추가
3. ETF 구입으로 $10,000 사용 (현금 수정)
→ 자동으로 총 자산액 계산!
```

### 방법 2: JSON 파일 직접 수정
`data/portfolio.json` 파일을 직접 편집할 수 있습니다.

**파일 위치:** `d:\Users\auto_stock_v2\data\portfolio.json`

**형식:**
```json
{
  "QYLD": {
    "name": "글로벌엑스 나스닥100 커버드콜",
    "holdings": [
      {
        "quantity": 100,
        "purchase_price": 45.50,
        "purchase_date": "2024-01-15",
        "total_invested": 4550.0,
        "added_date": "2024-01-15 10:30:00"
      }
    ]
  }
}
```

**항목 설명:**
- `QYLD`: ETF 심볼
- `name`: ETF 이름
- `quantity`: 보유 개수
- `purchase_price`: 매입가 (단가)
- `purchase_date`: 매입 날짜
- `total_invested`: 총 투자액 (개수 × 매입가)

### 방법 3: Python 코드로 추가

```python
from portfolio_manager import PortfolioManager

pm = PortfolioManager()

# 자산 추가
pm.add_asset("QYLD", "글로벌엑스 나스닥100 커버드콜", 100, 45.50, "2024-01-15")
pm.add_asset("SCHD", "슈왑 미국 배당주", 50, 80.00, "2024-02-20")

# 포트폴리오 보기
pm.print_portfolio()
```

## 🔧 고급 기능

### 자산 수정하기
가격을 잘못 입력했거나 정정이 필요한 경우:
```python
pm.update_asset("QYLD", 0, quantity=110, purchase_price=46.00)
```

### 자산 제거하기
```python
pm.remove_asset("QYLD", 0)  # 특정 보유 기록 제거
pm.remove_asset("QYLD")     # 전체 제거
```

### 특정 자산 정보 조회
```python
info = pm.get_asset_info("QYLD")
print(f"수량: {info['total_quantity']}")
print(f"평균가: ${info['average_price']}")
```

## ❓ FAQ

### Q: 현금은 어떻게 관리하나요?
A: 포트폴리오 관리 도구의 "5. 💰 현금 관리" 메뉴에서:
- 초기 자금, 월급, 배당금 등 현금 추가
- 펀드 인출이나 구입으로 인한 현금 변동 기록
- 여러 번의 거래도 각각 기록 가능

### Q: 총 자산액을 계산할 때 현금이 포함되나요?
A: 네! 포트폴리오 보기에서 다음이 자동 계산됩니다:
- **ETF 투자액**: ETF 자산의 총액
- **현금 잔고**: 보유 현금의 총액
- **총 자산액**: ETF + 현금

### Q: 현금을 음수로 입력할 수 있나요?
A: 네! 예를 들어:
- 초기 자금 $50,000 추가
- 주식 구입으로 -$10,000 입력
- 배당금으로 +$500 추가
→ 자동으로 네팅된 현금액 계산

### Q: 실시간 수익률을 볼 수 있나요?
A: 곧 추가될 예정입니다. main.py에서 현재 가격과 연동하여 수익률을 자동 계산하는 기능을 개발 중입니다.

### Q: 여러 개의 거래 기록을 관리할 수 있나요?
A: 네! 같은 ETF도 여러 번 구입한 기록을 모두 저장할 수 있습니다. 현금도 마찬가지로 여러 거래를 각각 기록할 수 있습니다.

### Q: JSON 파일 형식이 틀렸을 때는?
A: 파일이 손상되면 `data/portfolio.json.bak`에서 복구하거나, CLI 도구를 다시 사용하세요.
